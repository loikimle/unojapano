== Description ==

MailChimp synchronization for user registration plugin.

Get [free support](https://wpeverest.com/support-forum/)


### Features And Options:
* Mailchimp integration with registration form
* Field mapping (with mailchimp list)
* Auto field synchronization with mailchimp list feature
* Incredible Support
* Well Documented
* Translation ready
