PostX Pro is a Gutenberg blocks plugin for latest post blocks, post grid, post list, popular post and post slider blocks.

== Description ==

PostX Pro is a Gutenberg post block plugins for creating beautiful Gutenberg post grid blocks, post listing blocks, post slider blocks and post carousel blocks within a few seconds.
